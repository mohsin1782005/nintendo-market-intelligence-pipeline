# Project AETHELRED: Nintendo Platform Competitive Market Intelligence

An end-to-end analytics platform tracking 1,026 Nintendo titles for STRATOS Gaming Group. This project highlights a full data pipeline from raw data ingestion to interactive business intelligence and mobile alert architecture.

## 🛠️ Tech Stack & Skills
* **Data Engineering:** Python (Pandas/NumPy), SQL, ETL Pipeline Architecture
* **Business Intelligence:** Tableau Desktop, Tableau Prep
* **Analytics:** KPI Metric Engineering, Trend Forecasting

## 🚀 Key Deliverables
* **Automated Ingestion Pipeline:** Structured data mapping raw game logs into optimized, analytics-ready schemas.
* **Market Intelligence Dashboard:** Centralized executive view tracking performance via critical metrics (Meta Score Coverage, User Score Trends, and ESRB breakdowns).
* **Mobile Alert System Architecture:** Blueprint enabling real-time automated metric alerts pushed directly to mobile devices.

## 📊 Key Insights Captured
* Successfully mapped historical release velocity across major platforms from 1996 to 2021.
* Engineered an aggregated scoring matrix assessing target player demographics by user-score performance.
